11. While, For Loops and Matrix

1] While Loop : A while loop running as long as its condition is True.
python checks the condition before every iteration. the moment it becomes false, the loop stops

2] FOR loop
for loop is used to iterate over a sequence - a range of numbers, a string, a list
and so on. unlike while, you don't manage a counter manually - Python handles it for you.

for variable in sequence # this block runs for each item in the sequence


# looping over a range of numbers
for i in range(1, 6):
    print(i) 

# output: 1 2 3 4 5



12. Controlled statements : 

1) BREAK Statement :

 Break immediately stops the loop and the exits it, even if the condition
 is still true or there are items left in the sequence.

e,g 

num = 1
while num <= 10
   if num == 5:
       break        #stops the loop when num reaches 5
    print(num)

#output: 1 2 3 4

2) CONTINUE Statement :
 
 Continue skips the rest of the current iteration and jumps straight to the next one. 
 the loop does not stop-it just skips that particular cycle.

e.g 

for i in range(1, 11):
    if i % 2 == 0:
        continue          # will skip even numbers
    print(i)            

# output: 1 3 5 7 9


3] Nested Loops:
 A nested loop is a loop inside another loop. The inner loop completes all its 
 iteration for every single iteration of the outer loop

for outer in range(1, 4):
    for inner in range(1, 4):
        print(f"outer={outer}, inner={inner}")

15. return Statement

The return statement sends a value back to whoever called the function.
without return, a function just does something but gives nothing back.
with return, you can use the result in the rest of your program.

# with return - result can be stored and used
def add(a, b):
    return a + b

result = add(10, 5)
print(result)
print(add(3, 7) * 2)


Note : Once python hits a return statement, the function stops
immediately, any code return inside the function will never run.

18. Lambda Function 
 
 A lambda function is small anonymous function writtern in a single line, 
 it is useful when you need a simple function for a short period and do not want to formally define it with def.

 e.g 
 # normal function
 def square(n):
     return n * n


 #same thing as a lambda
 square = lambda n: n * n

 print(square(5))         #25

 is_adult = lambda age: True if age >= 18 else False
 print(is_adult(18))   #True


 19. Lists in Python

 A variable stores one value at a time.
 So A list is a collection that stores multiple values in a single variable, kept in a specific order

 student = ["Tejal", "Sarthak", "Advika"]

 all thre names stored in one variable. easy to manage , scale and read

 key properties:
  1) ordered :- Items stays in the exact order you put them in. the first item always first
  2) mutable :- Can change, add, remove items even after the list created.
  3) Allows Duplicates : the same value can appear more than once - list do not enforce uniqueness
  4) Any data type : A single list can hold intergers, strigs, boolean or even other Lists

 **** creating a list :

 1) Empty List : start with an empty list and add items later
    my_list = []

 2) Numbers (integers/Floats) : Stores a sequence of numeric values
    marks = [29, 21, 30, 81, 87]

 3) Text (Strings) : Hold a collection of words or phrases
    cities = ["mumbai", "chennai", "delhi"]

 4) Mixed data types : Lists can contain different types of data in one go
    profile = ["Sarthak", 29, 28.1, True]

 5) Lists with Duplicates : Python lists readily allow repeated values

 6) Checking Type and length : use type() to confirm it's a list
                               len() to get the item count.

    print(type(marks))         # <class 'list'>
    print(len(marks))


   **** List Operators: Joining and Repeating  --

   1) Joining list with '+'  :- The '+' Operator concatenated (joins) two or more lists,
                                creating an entirely new list. The original list remain unaffected.

        a = [10, 30]
        b = [29, 11, 3]
        combined = a + b
        print(combined)               # output : [10, 30, 29, 11, 3]

    2) Repeating Lists with '*' :- The '*' operator repeats the elements of a list a specified number of times,
                                   returning a new list. the multiplier must be an integer.

        items = ["book", "pen"]
        repeated = items * 2
        print(repeated)                 #output: ["book", "pen", "book", "pen"]

    

    **** Built-in functions on lists 

    1) len() - Get length   
    returns the number of elements present in the list

    2) min() - Smallest value
    returns the Smallest item from a list of numbers

    3) max() - Largest value
    returns the largest item from a list of numbers

    4) sum() - Sum of elements
    returns the sum of all numeric elements in the list

    5) sorted() - Create sorted list
    returns a new list containing all items from the iterable in ascending order. 

      calculating average marks - combining sum() and len() provides a simple way to calculate the average of numerical data in a list
      
      marks = [1, 2, 3, 4]
      average = sum(marks) / len(marks)


    **** Accessing list elements : indexing

    Occupies specific position its called 'Index'
    zero-based indexing, first element is at 0 index.

    positive indexing: left to right :-

    x = ["a", "b", "c", "d"]

    negative indexing: right to left
    x = ["a", "b", "c", "d"]
    #     -4   -3   -2   -1
